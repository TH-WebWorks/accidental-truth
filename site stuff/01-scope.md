## In Scope

Deliverables included in the $750 engagement (per proposal and execution directives):

| Item | Details |
|------|---------|
| Site | New website on accidentaltruths.com |
| Structure | Single-page build |
| Content migration | Pull existing content from MUFON page |
| Film materials | Insert new film materials at top |
| Merchandise section | Shopify or WooCommerce embed included in scope as lightweight integration (no full store buildout). |
| Assets | Use supplied graphics and media |
| Design | Collaborative design phase |
| Thematic positioning | Aligned to documentary subject matter |
| Domain | Bring domain live |
| Technical terms | $750 flat rate; estimated 2–3 day development window post-design approval |

**Asset checklist:** See [05-content-inventory.md](05-content-inventory.md) for content and asset status.

**Hosting:** See [09-domain-dns-plan.md](09-domain-dns-plan.md) for domain and hosting configuration.

## Explicit Exclusions

| Item | Notes |
|------|-------|
| MUFON WordPress integration | Build is standalone; not within MUFON WordPress environment. MUFON serves as distribution channel, not primary property. |
| Multi-page expansion | Single-page structure for initial launch. Expandability is a future-phase consideration. |
| E-commerce buildout | Merchandise section is an embed (Shopify/WooCommerce); full store setup is out of scope unless separately scoped. |
