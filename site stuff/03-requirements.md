# Requirements

**Project:** Accidental Truth
**Client:** Ron James
**Document Owner:** T&H WebWorks
**Last Updated:** 02/12/2026

---

## Purpose

This document defines functional and technical requirements derived from the approved scope. It serves as the build-ready specification before design and development phases.

---

## Functional Requirements

- **Single-page structure** — All content on one scrollable page; no internal navigation to separate pages.
- **New film section at top** — Primary placement for the new film; first content block below header.
- **Original film section below** — Secondary placement; content migrated from MUFON page.
- **Merchandise embed section** — Shopify-ready, modular embed; lightweight integration (no full store buildout).
- **Footer** — Credits and links.
- **Mobile responsive layout** — Usable across viewport sizes.

---

## Content Requirements

### New Film

- Title
- Synopsis
- Trailer embed
- Visuals
- CTA (call-to-action)

### Original Film

- Content pulled from existing MUFON page

### Assets

- All client-supplied graphics and media

---

## Technical Requirements

- **Domain:** accidentaltruths.com
- **DNS:** Via Directnic
- **SSL:** Required
- **Architecture:** Standalone site; no MUFON WordPress dependency
- **Performance:** Fast load
- **SEO:** Basic setup — meta tags, Open Graph, canonical URL
